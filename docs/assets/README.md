# Content Assets

Generated launch assets for Group Leveling.

Run:

```bash
npm run assets:launch
```

Files:

- `group-leveling-social-card.png`: general GitHub/social preview.
- `group-leveling-architecture.png`: high-level architecture visual.
- `group-leveling-demo-flow.png`: 30-second product story visual.
- `group-leveling-invite-card.png`: invite and trusted-team visual.
- Matching `.svg` files: editable source images generated by the same script.
- `launch-copy.md`: short copy blocks for posts and screenshots.

These assets avoid using private host URLs, tokens, real teammate names, or local filesystem paths.
